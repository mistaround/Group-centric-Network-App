# Group-centric Social Network App

The project ultilizes MERN stack:

MongoDB, Express.Js, React.Js, Node.Js

For docs, please refer to project wiki

## Implemented features:

- Register/Login
- Account Management
- Group Management
- Multi-media
- Activity Feed
- Post & Comment
- Live Chat
- Real-time Notification
- Group Recommendation

## To build

### Frontend

```shell
cd ./client
npm install
npm run build
npm start
```

### Backend

```shell
npm install
npm start
```

### Socket

```shell
npm install
npm start
```
